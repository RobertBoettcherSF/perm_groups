# Permutation Groups in Ada SPARK

**Implementation of Donald E. Knuth's Algorithms Aₖ(π) and Bₖ(π)**

This repository contains a complete Ada SPARK implementation of Donald E. Knuth's algorithms for computing strong generators and transversal systems of permutation groups, as described in *The Art of Computer Programming*, Volume 2, Section 4.6.3.

## Overview

This implementation provides:

- **Algorithm Aₖ(π)**: Appends a new permutation π to the strong generating set T(k)
- **Algorithm Bₖ(π)**: Ensures permutation π is in the group Γ(k)
- **Membership Testing**: Recursive function to check if a permutation is in the group generated by the current Sigma system
- **Complete SPARK Verification**: All code is SPARK 2022 compatible with formal verification support

## Features

### Core Algorithms

1. **Strong Generating Set Computation** (`Compute_Strong_Generators`)
   - Takes an array of generator permutations
   - Computes the complete strong generating set using Knuth's algorithms
   - Returns Sigma (transversal system) and T (strong generators)

2. **Membership Testing** (`Is_Member`)
   - Recursively checks if a permutation is in the group generated by Sigma
   - Uses Subprogram_Variant to prove termination
   - Implements Knuth's membership test from Algorithm 4.6.3A

3. **Individual Generator Addition** (`Add_Generator`)
   - Adds a single generator to the existing group
   - Maintains the invariants of the Sigma and T structures

### Data Structures

- **Permutation**: Array type mapping Index to Index (bijections)
- **Permutation_Vector**: SPARK-compatible vector type (replaces Ada.Containers.Vectors)
  - Fixed-size array with length counter
  - Helper functions: Vector_Length, Vector_Append, Vector_Element, Vector_Clear
- **Sigma_Type**: 2D array of Permutation_Vectors for the transversal system Σ(k,j)
- **T_Type**: Array of Permutation_Vectors for strong generators T(k)

### Utility Functions

- **Identity**: Returns the identity permutation
- **Multiply**: Computes permutation composition (Left ∘ Right)
- **Inverse**: Computes the inverse of a permutation
- **Is_Identity**: Checks if a permutation is the identity
- **Create_Transposition**: Creates a transposition (swap of two elements)
- **Create_Cycle**: Creates a cycle permutation from a list of elements
- **"=" Operator**: Checks equality of two permutations

### Example Usage in Ada Code

```ada
with Permutations;

procedure Test_Permutation_Groups is
   use Permutations;
   
   -- Define some permutations
   Gen1 : Permutation := Create_Transposition(1, 2);
   Gen2 : Permutation := Create_Transposition(2, 3);
   
   -- Array of generators
   Generators : Generator_Array(1 .. 2) := (Gen1, Gen2);
   
   -- Compute strong generating set
   Sigma : Sigma_Type;
   T : T_Type;
   
begin
   Compute_Strong_Generators(Generators, Sigma, T);
   
   -- Test membership
   Test_Perm : Permutation := Multiply(Gen1, Gen2);
   if Is_Member(Test_Perm, Index'Last, Sigma) then
      -- Test_Perm is in the group
      null;
   end if;
end Test_Permutation_Groups;
```

## Implementation Details

### SPARK Compatibility

This implementation addresses several SPARK-specific challenges:

1. **No Ada.Containers.Vectors**: Uses custom `Permutation_Vector` type with fixed-size arrays
2. **Recursion Termination**: Uses `Subprogram_Variant` aspect for recursive functions
3. **Loop Invariants**: All loops include `pragma Loop_Invariant` for SPARK verification
4. **Explicit Initialization**: All functions explicitly initialize their results
5. **Type Safety**: Proper type conversions between Positive, Integer, and custom types

### Algorithm Complexity

Knuth's algorithms have the following characteristics:

- **Algorithm Aₖ(π)**: O(|T(k)| × |Σ(k)|) complexity
- **Algorithm Bₖ(π)**: Recursive, calls Algorithm Aₖ₋₁
- **Is_Member**: Recursive with depth at most Max_Size

The implementation uses:
- Maximum permutation size: 100 (Max_Size)
- Maximum vector size: 1000 (Max_Vector_Size)

### Mathematical Background

From Knuth, TAOCP Volume 2, Section 4.6.3:

- **Γ(k)**: The subgroup of permutations that fix points k+1, k+2, ..., n
- **Σ(k,j)**: The transversal system for Γ(k), containing permutations mapping k to j
- **T(k)**: The strong generating set for Γ(k)

The algorithms maintain the invariant:
```
Γ(k) = <T(k)>  (the group generated by T(k))
```
